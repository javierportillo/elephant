### Elephant Pass

Create and access password store

#### Example snippets

```toml
[[pass]]
keywords = ["personal"]
name = "Personal Store"
path = "/home/javier/.password-store"
key = "password@store"

[[pass]]
keywords = ["work"]
name = "Work Store"
path = "/home/javier/.password-store-work"
key = "workemail@domain.com"
```
